Файлы с раширеним .html открываются браузером.

Все файлы .html состоят из тегов.

  • Структура страницы

Справочник тегов.

1. Теги для текста

  • p - абзац от англ. paragraph
  • h1..h6 - заголовки от англ. heading
  • ul/ol - маркированный и нумерованнный списки от англ. unordered ordered list, li - пункты списков от англ. List item
  • a - якорь от англ. anchor Подробнее

2. Теги для выделения текста

  • b - жирный от bold
  • i - курсивный от italic
  • del - перечеркнутый от delete

3. Теги для графических элементов

  • hr - линия от horizontal rule
  • img - картинка от англ. image

4. Теги, задающие структуру страницы

  • !DOCTYPE html - не тег а инструкция для браузера
  • html - заключает в себя всё содержимое веб-страницы
  • head - для хранения служебной информации о странице
  • body - содержимое html документа
  • header - раздел шапки сайта (логотип, меню, вход в ЛК)
  • main - весь основной контент страницы
  • section - раздел веб-страницы
  • footer - раздел подвала (контакты, соц. сети copyright)
  • nav - раздел с меню
  • div универсальный блок
